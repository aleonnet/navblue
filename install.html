Nenhuma porta selecionada

Se você não selecionou uma porta porque o seu device não apareceu na lista, tente os passos abaixo:

  1. Confira se o device está conectado a este computador (o que roda o navegador com esta página).
  2. A maioria dos devices tem um LED aceso quando ligado. Se o seu tiver, confira se está aceso.
  3. Confira se o cabo USB transmite dados — não é um cabo só de energia.
  4. No Linux, garanta que o seu usuário está no grupo dialout para ter permissão de acesso ao device. sudo usermod -a -G dialout SeuUsuario Pode ser preciso sair e entrar de novo (ou reiniciar) para ativar o novo grupo.
  5. Confira se os drivers corretos estão instalados. Abaixo, os drivers dos chips mais comuns em devices ESP:
    • CP2102 drivers: Windows e Mac
    • CH342, CH343, CH9102 drivers: Windows, Mac (baixe pelo ícone de nuvem azul na página do fabricante)
    • CH340, CH341 drivers: Windows, Mac (baixe pelo ícone de nuvem azul na página do fabricante)
